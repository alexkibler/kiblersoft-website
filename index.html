What we do

As a company, Kiblersoft specializes in .NET Web Applications, hosted on Microsoft's Azure platform. We mostly create passion projects and new ideas.

5E Angular Spellbook

The spellbook is an application for Dungeons and Dragons players, allowing them to save spells to a personal portfolio, saving time usually wasted flipping through pages.

Material Bowling Scorekeeper

Currently in development, Material Bowling Scorekeeper will be a beautifully designed modern scorekeeping app for bowlers.

My Boardgame Collection

Also currently in development, MBC is an application that utilizes BoardGameGeek's API to allow users to store their board game collection and view setup and playtimes, total value, and more

Material Bowling Mobile

Also in development is the mobile version of Material Bowling, for Android/iOS/Windows Phone
